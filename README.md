# Mini Football 4v4 Prototype

Playable web prototype for a casual 3D top-down mini football game.

## Run

Open the local server URL:

```text
http://127.0.0.1:5174/
```

The prototype uses Three.js from CDN, so the browser needs internet access for the first load.

## Mobile install

The game is configured as an installable PWA.

Android:

1. Open the GitHub Pages link in Chrome.
2. Tap the browser menu.
3. Choose **Add to Home screen** or **Install app**.

iPhone:

1. Open the GitHub Pages link in Safari.
2. Tap Share.
3. Choose **Add to Home Screen**.

## Controls

- Move: `WASD` or arrow keys
- Kick: `Space`
- Switch to nearest blue outfield player: `Q`
- Mobile: left virtual stick, right action buttons

## Prototype Scope

- 4v4 match
- Top-down 3D field
- One controlled blue player
- Simple teammate and opponent AI
- Ball collisions, goals, score, match timer
- Mobile-friendly touch controls

## Monetization Direction

Good first monetization targets for this kind of casual mobile game:

- rewarded ads for extra coins, second chances, cosmetics
- interstitial ads between short matches
- cosmetic purchases: kits, balls, stadium skins, player trails
- progression pass with daily challenges

The next useful milestone is a stronger game loop: short matches, coins, upgrades, cosmetics, and a post-match reward screen.
